# SQL Querying and Data Modification

This project demonstrates core SQL operations including querying, aggregation, joins, and data modification.

## Topics Covered
- Filtering and sorting (WHERE, ORDER BY)
- Aggregation (COUNT, AVG)
- Grouping and HAVING
- Joins and multi-table queries
- Subqueries
- Data modification (INSERT, UPDATE, DELETE)

## Key Concepts
- Writing efficient queries across multiple tables
- Using subqueries for complex filtering
- Modifying relational data while maintaining integrity

## Files
- `queries.sql` — SELECT queries and aggregations
- `modifications.sql` — INSERT, UPDATE, DELETE operations
