# Advanced Database Systems (CS 582)

This repository contains implementations of advanced database concepts across SQL, NoSQL, and database programming.

## Topics Covered
- SQL querying, aggregation, and subqueries
- Constraints, triggers, and assertions
- Views, indexing, and performance optimization
- Stored procedures and database programming (ODBC, JDBC, PHP)
- OLAP and data cube operations

## Highlights

### Query Engine (Python)
- Implemented selection and projection operations without SQL parsing

### SQL Systems
- Complex queries with grouping, HAVING, and subqueries
- Schema constraints and trigger-based validation

### Database Programming
- ODBC (C), JDBC (Java), PHP integration with MySQL
- Stored procedure creation and execution

### OLAP
- Implemented GROUP BY CUBE for multi-dimensional aggregation
