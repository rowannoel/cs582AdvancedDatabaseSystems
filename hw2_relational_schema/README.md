# Relational Schema Design (SQL)

This project implements a simple relational database schema using MySQL.

## Features
- Database and table creation
- Primary and foreign key constraints
- Data insertion using subqueries
- Basic SELECT queries for data retrieval
- Referential integrity testing

## Schema
- Students(student_id, name, major)
- Enrollments(enrollment_id, student_id, course)

## Key Concepts
- Foreign key relationships between tables
- Ensuring referential integrity
- Querying relational data
- Testing constraint violations

## Files
- `schema.sql` — full database setup and queries
